# TICK Stack in Docker

## Background
This project initially created for isolating metric monitoring system from host server (in my case is NAS server). Other feature and integration may be added in later time. Feel free to fork this repo and experimenting with your use-case.

## Prerequisite
  - Docker Engine (version 18.02.0 or later)

## Usage
`docker stack deploy -c ./docker-compose.yml <your service name>`

## Contributing
If you want to contribute to this project, simply fork and make your changes on it. Then, make a PR (Pull Request) and I will review your changes.
